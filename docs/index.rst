PyArg
=====

Getting started with PyArg
==========================

Visualisation
-------------

If you only want to use the visualisation of PyArg, go to
https://pyarg.npai.science.uu.nl/.

Installing the package
----------------------

For installing the PyArg package, make sure that you have the latest
version of pip installed and run:

::

   pip install python-argumentation

Accessing the source code
-------------------------

Alternatively, if you wish to inspect or contribute to the source code,
the code is available on https://github.com/DaphneOdekerken/PyArg.

::

   git clone https://github.com/DaphneOdekerken/PyArg.git

In case you run into import issues, make sure to add the src folder to
your PYTHONPATH.
